# Forex Dashboard - pet project:
Fullstack web app. Forex traders dashboard.

## Build with:
1. React with NextJS
   - NextJS API router
   - React custom hooks
2. Typescript
3. Tailwind CSS
4. Rechart

### Code formatting:
1. TSlint
2. ESLint
3. Prettier

#### Run
clone => npm start => npm run dev => project will start on http://localhost:3000
